# 👋 欢迎使用Langflow

Langflow 是一种全新的、通过可视化的方式来构建、迭代和部署AI应用的大模型应用开发平台。

它提供直观的界面，使开发人员通过简单的操作 AI 组件，就能够快速把他们的想法原型化并转化为强大的真实世界解决方案。
